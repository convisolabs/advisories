[CONVISO-22-003] - FortiADC / FortiDDoS-F - Restricted user able to modify system files via shell access

1. Advisory Information

   Conviso Advisory ID: CONVISO-22-003
   CVE ID: CVE-2021-43076
   CVSSv3 Score: 6.2
   Severity: Medium
   Date: 2022-09-06

2. Affected Components

   FortiDDoS-F:

   * Version 6.3.0

   FortiADC:

   * Version 6.2.0 through 6.2.1
   * Version 6.1.0 through 6.1.5
   * All versions of 6.0
   * All versions of 5.4
   * All versions of 5.3

   Not affected:

   * FortiDDoS-F 6.2.x
   * FortiDDoS-F 6.1.x
   * FortiADC 7.1.x
   * FortiADC 7.0.x
   * FortiADC 5.2.x
   * FortiADC 5.1.x

3. Description

   An improper privilege management vulnerability [CWE-269] in FortiADC and FortiDDoS-F
   may allow a remote authenticated attacker with a restricted user profile to modify
   system files using shell access.

4. Details

   * The vulnerability affects privilege management controls in FortiADC and FortiDDoS-F.
   * A restricted authenticated user can obtain shell access to the underlying system.
   * Authorization restrictions applied to low-privileged accounts are not properly enforced.
   * Through shell access, an authenticated user may modify system files beyond the permissions
     intended for the assigned role.
   * Successful exploitation may compromise the integrity of the appliance and allow unauthorized
     changes to system-level resources.

5. Solution

   Upgrade to one of the following fixed versions:

   FortiADC:

   * Version 6.2.2 or above
   * Version 7.0.0 or above

   FortiDDoS-F:

   * Version 6.3.1 or above
   * Version 6.2.3 or above
   * Version 6.1.5 or above
